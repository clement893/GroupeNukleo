# Plan de création — Pages détail des 4 services

Référence : maquette « Studio créatif » (fond clair, titre + tagline, hero image, section 2 colonnes avec nav secondaire, bloc « Extensions créatives », grille d’offres, équipe, processus, CTA, footer Fort.).

**Pages concernées :**
1. **Studio créatif** — `/services/studio` (NukleoStudio)
2. **Lab technologique** — `/services/tech` (NukleoTech)
3. **Agence Comm & Marketing** — `/services/agency` (NukleoAgency)
4. **Transition numérique** — `/services/consulting` (NukleoConsulting)

---

## 1. Structure commune (toutes les pages)

| Ordre | Section | Description |
|-------|---------|-------------|
| 1 | **Header** | Global (PageLayout) : logo nukleo., bouton Contactez-nous, icône recherche/menu |
| 2 | **Titre page + tagline** | Titre principal (ex. « Studio créatif ») en gros, bold, bordeaux/noir ; sous-titre en gris (une ligne) |
| 3 | **Hero image** | Grande image rectangulaire pleine largeur (thématique du service) |
| 4 | **Bloc 2 colonnes (nav + contenu)** | Gauche : liste verticale de sous-services (lien actif mis en avant). Droite : titre du sous-service, paragraphe, tags (petites cartes), optionnel élément graphique flottant |
| 5 | **Bloc « Extensions » / offre liée** | Image à gauche, texte à droite : titre, paragraphe, tags (ex. « Extensions créatives » pour Studio) |
| 6 | **Grille d’offres** | 4×2 ou 3×2 cartes : titre + courte description (ex. Branding & Identité, UI/UX Design, …) |
| 7 | **Section équipe** | Titre (ex. « Les créatifs derrière vos projets »), paragraphe de présentation |
| 8 | **Section processus** | 4 cartes horizontales : étape (Discovery, Concept, Design, Livraison) avec titre + courte description |
| 9 | **CTA** | Fond lavande ; titre « Prêt à performer ? » ; bouton violet « Contactez-nous » + flèche |
| 10 | **Footer** | Séparateur « Fort. » puis footer global (logo, liens Accueil / Services / Projets / Contact, liste des services, contact, icône LinkedIn) |

---

## 2. Palette et styles

- **Fond page** : blanc ou off-white `#FAFAF9`
- **Titre principal** : bordeaux `#5A1E29`, très gros, bold
- **Tagline** : gris `#6b7280`
- **Labels / sous-titres** : bordeaux ou gris foncé
- **Cartes** : fond blanc, coins arrondis, ombre légère
- **CTA** : fond lavande `#F5F3F8`, bouton violet `#6B4BEA`
- **Séparateur Fort.** : texte `#5A1E29`, très gros
